# Mobile App

Flutter mobile application with clean architecture pattern.

## Architecture

- **l1_ui/**: UI layer (screens, widgets, viewmodels)
- **l2_domain/**: Domain models and business logic
- **l3_service/**: Service layer (API clients, providers)

## Setup

```bash
# Install dependencies
flutter pub get

# Run the app
flutter run
```
