# Exercise Type Icons

Place icon files for each exercise type in this folder.

## File Naming Convention
Use the exercise type ID as the filename:
- `all.png` - All exercises
- `chest.png` - Chest exercises  
- `back.png` - Back exercises
- `shoulders.png` - Shoulder exercises
- `legs.png` - Leg exercises
- `arms.png` - Arm exercises
- `core.png` - Core exercises
- `cardio.png` - Cardio exercises

## Icon Specifications
- **Format**: PNG with transparency
- **Size**: 48Ã—48px (will be scaled as needed)
- **Style**: Simple, bold, high contrast icons
- **Color**: Single color (will be tinted by app theme)
  - Use white/light color for the icon shape
  - Transparent background
  - App will apply lime green tint for selected state

## Usage
These icons are used in:
- Exercise Filter Type Page (grid tiles)
- Filter button (if applicable)

Icons should match the Bold Studio Aesthetic:
- Bold, geometric shapes
- High contrast
- Minimal detail
- Clear at small sizes
