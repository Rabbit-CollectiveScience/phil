# Backend API

Next.js backend API with clean architecture pattern.

## Architecture

- **l2_domain/**: Domain models and use case controllers
- **l3_data/**: Data access layer with repositories and adapters
- **l4_infra/**: Infrastructure layer (config, storage, external services)

## Setup

```bash
# Install dependencies
npm install

# Set up environment variables
cp .env.example .env.local

# Run development server
npm run dev
```
